# philosophers

## Usage
```
$ make
$ ./philo <philosopher num> <time to die> <time to eat> <time to sleep>
```
## Example
- Not a single philosopher will die.
```
$ ./philo 4 410 200 200
```
- Philosophers will die.
```
$ ./philo 4 310 200 100
```
